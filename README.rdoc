== README

TECHNOLOGIES USED:

- Ruby on Rails (Ruby 4)
- Html5
- CSS3
- Javascript
- Heroku
- Git

LINKS:

- Trello  https://trello.com/b/0jN7nRb7/catering-crud-app
- Heroku  https://cater-hambones.herokuapp.com/


INSTALLATION INSTRUCTIONS:

1. Deploy app using Heroku link provided above
2. Application dependent on information provided through Sign Up or Log In
3. Sign up as a customer providing details required
4. Start an order
5. Provide order details
6. Add menu items to order
7. Review order
8. Submit order
9. Receive email confirmation with order details
